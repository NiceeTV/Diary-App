# Denník App

Jednoduchá aplikácia na denné zaznamenávanie aktivít, cieľov a reflexie.

## Funkcie
- ✅ Zaznamenávanie aktivít cez timeline
- ✅ Sledovanie cieľov a reflexie
- ✅ Štatistiky (vstávanie, spánok, screen time)
- ✅ Interaktívna timeline s klikateľnou čiarou
- ✅ Upload hlasoviek a súborov

## Technológie
- Python + Flask (backend)
- HTML + CSS + JavaScript (frontend)
- JSON (úložisko)

git clone https://github.com/NiceeTV/Diary-App.git
